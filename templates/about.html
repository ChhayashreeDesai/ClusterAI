Machine Learning Algorithms

K-Means Clustering

Partitions data into K clusters where each observation belongs to the cluster with the nearest mean (centroid).

Advantages:
  • Fast and efficient for large datasets
  • Works well with spherical clusters
  • Simple to understand and implement
from sklearn.cluster import KMeans
kmeans = KMeans(n_clusters=k, random_state=42)
clusters = kmeans.fit_predict(scaled_data)
Agglomerative Clustering

Hierarchical clustering that builds nested clusters by merging or splitting them successively.

Advantages:
  • No need to specify number of clusters upfront
  • Creates a hierarchy of clusters
  • Works with any distance metric
from sklearn.cluster import AgglomerativeClustering
agg = AgglomerativeClustering(n_clusters=k)
clusters = agg.fit_predict(scaled_data)
DBSCAN

Density-Based Spatial Clustering finds clusters of varying shapes and identifies outliers as noise.

Advantages:
  • Finds clusters of arbitrary shape
  • Automatically determines cluster count
  • Robust to outliers
from sklearn.cluster import DBSCAN
dbscan = DBSCAN(eps=0.5, min_samples=5)
clusters = dbscan.fit_predict(scaled_data)
Elbow Method + Silhouette

Combined approach to find optimal number of clusters using mathematical optimization techniques.

Metrics Used:
  • Within-Cluster Sum of Squares (WCSS)
  • Silhouette Score
  • Calinski-Harabasz Index
from kneed import KneeLocator
knee = KneeLocator(k_range, wcss, curve='convex')
optimal_k = knee.elbow

Evaluation Metrics

Silhouette Score

Range: -1 to 1 (Higher is better)

Measures how similar an object is to its own cluster compared to other clusters. A high value indicates that the object is well matched to its own cluster.

Formula: (b - a) / max(a, b)

Where a = mean distance to points in same cluster, b = mean distance to nearest cluster

Calinski-Harabasz Index

Range: 0 to ∞ (Higher is better)

Ratio of the sum of between-clusters dispersion and inter-cluster dispersion. Higher values indicate better defined clusters.

Also known as: Variance Ratio Criterion

Evaluates cluster validity based on average between and within cluster sum of squares

Davies-Bouldin Index

Range: 0 to ∞ (Lower is better)

Average similarity measure of each cluster with its most similar cluster. Lower values indicate better clustering.

Measures: Cluster separation and compactness

Based on ratio of within-cluster distances to between-cluster distances
